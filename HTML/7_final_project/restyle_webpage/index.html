Restyle a Webpage

Below is a webpage. You only have access to editing the CSS. There is a set of things that you need to do to the webpage, just using the CSS, which are listed below:

  • Align all h1 elements to the right
  • Center the text for all the h2s
  • Make all links green
  • Make all h3s blue, with an orange background color, and set them to have a height of 250 pixels.
  • Center the text of elements with the "closing" id (ids use a #element, classes use .element)
  • Change the text color for the "opener" class to white, and set the background color to yellowgreen.
  • Make the tables take up 75% of the screen horizontally.
  • Make the top of the table a different color.
  • Add a background image to the "middle" class. Use 'image.png' as your path.

Feel free to use the internet to figure out anything that hasn't already been explained or that you forgot.
