Tags

HTML is made up of tags. Tags are written like this: <tag_name>...</tag_name>. Inside is whatever contents it has, usually either text, or more tags. One of the most common tags, which is used for text is the "p" tag (if you type in the box it will render):

There are also tags for title text, subtitle, or generally just any larger text that you might need.

Baggage

These elements are cool, but of course, everything cool comes with some baggage that must be setup in order for everything to work properly.
There are three important tags to learn:

  • <html> - Surrounds all of the code.
  • <head> - Surrounds the part where the <title> tag is.
  • <body> - Surrounds all of the tags from before, and more
  • <title> - Goes in the <head>and sets the title of the page, shown on the tab

  • The a tag: This is a link, meaning if you click on it it will go to another website, or page on your website. These "paths" act exactly like file paths on the computer, because they literally are linking to folders or (html) files on the computer.
