HTML Reference

Non-body elements

  • html - starts/ends an html file

  • head - Where you put the title and other elements you CAN'T see in the page

  • title - What appears in the tab at the top of the browser

Body elements

  • body - Holds all the visible elements in the page.

  • h1 - Heading 1, the largest title used probably for the title of the document that's actually on the page.

  • h2 - Second largest title, usually for sections in the document.

  • h3 - Title but for other purposes, possibly used if your sections have subsections.

  • p - Used for writing paragraphs with text, puts newlines before and after the text.

  • strong - Bold text.

  • i - Italic text.

  • a - Hyperlink

  • img - Image, what a surprise.
